# Code Of Conduct

This project expects professional, respectful collaboration.

## Expected Behavior

- Be direct without being hostile.
- Assume good intent, but accept correction when impact does not match intent.
- Keep discussions focused on the work and the evidence.
- Make space for maintainers and contributors with different backgrounds and experience levels.

## Unacceptable Behavior

- Harassment, threats, insults, or personal attacks.
- Sexualized language or attention.
- Publishing private information without permission.
- Repeated disruption of technical discussions after maintainers ask for a change in behavior.

## Enforcement

Maintainers may remove comments, close threads, block accounts, or take other proportionate action when this code of conduct is violated. Report concerns privately through the project maintainers or the repository owner organization.
